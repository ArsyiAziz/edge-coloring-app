# edge-coloring-app

This a project for demonstrating the coloring of edges on graph. Build using [Cytoscape.js](http://js.cytoscape.org) and [Tailwind CSS](http://tailwindcss.com)

## Features
- [x] Add Node
- [x] Add Edge by dragging a node to another node
- [ ] Delete Node
- [ ] Delete Edge
- [ ] Edge Coloring
